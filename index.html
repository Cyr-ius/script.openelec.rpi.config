OpenELEC RPi Config

Unofficial Raspberry Pi configuration add-on for OpenELEC

Download this project as a .zip file Download this project as a tar.gz file

This add-on makes it possible to edit the Raspberry Pi system configuration parameters in /flash/config.txt from the XBMC gui.

Changelog

Installation

  1. Install my XBMC repository from xbmc.repo.leopold.zip.
  2. Install OpenELEC RPi Config from Leopold's Add-ons under Services.

Usage

Currently only a subset of all possible properties can be configured. The complete list is shown below.

Overclocking

  • arm_freq
  • core_freq
  • sdram_freq
  • over_voltage
  • over_voltage_sdram
  • force_turbo
  • initial_turbo

Video

  • gpu_mem
  • hdmi_force_hotplug
  • hdmi_drive
  • hdmi_force_edid_audio
  • hdmi_pixel_encoding
  • sdtv_mode
  • sdtv_aspect
  • disable_overscan
  • overscan_scale
  • overscan_left
  • overscan_right
  • overscan_top
  • overscan_bottom

Misc

  • decode_MPG2
  • decode_WVC1
  • hdmi_ignore_cec
  • disable_splash

For more information see http://elinux.org/RPiconfig.

Reporting Issues

If you get a script error or other problem please raise an issue. If you can reproduce the problem then please include a debug log, by following these steps:

  • Turn on XBMC debugging.
  • Turn on the 'Debug' option in the add-on advanced settings.
  • Restart XBMC to start a clean log.
  • Run the add-on again to reproduce the error.
  • Upload the debug log. There are a couple of ways:
    1. Log in via ssh and run pastebinit ~/.xbmc/temp/xbmc.log
    2. Use the add-on "XBMC Log Uploader".
